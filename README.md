# aws-simple-cost-insight
Visualize and compare AWS costs across multiple accounts using exported CSVs.

## 🔒 データプライバシー
**完全ローカル処理**: アップロードしたCSVデータはブラウザ内でのみ処理され、外部に送信されることは一切ありません。
